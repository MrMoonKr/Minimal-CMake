# ch2/part-3

Adding resizable array functionality in a separate file.

## Visual Studio Code Debugging

Open Visual Studio Code from `ch2/part-3`

```bash
cd ch2/part-3
code .

# or

code ch2/part-3
```

## Commands

```bash
# start
cd ch2/part-3
# configure
cmake -B build -G "Ninja Multi-Config"
# build
cmake --build build
# run
./build/Debug/minimal-cmake_game-of-life
```

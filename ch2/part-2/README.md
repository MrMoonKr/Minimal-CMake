# ch2/part-2

Introduces initial _Game of Life_ console application.

## Visual Studio Code Debugging

Open Visual Studio Code from `ch2/part-2`

```bash
cd ch2/part-2
code .

# or

code ch2/part-2
```

## Commands

```bash
# start
cd ch2/part-2
# configure
cmake -B build -G "Ninja Multi-Config"
# build
cmake --build build
# run
./build/Debug/minimal-cmake_game-of-life
```

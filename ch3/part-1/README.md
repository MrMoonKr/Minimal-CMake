## Visual Studio Code Debugging

Open Visual Studio Code from `ch3/part-1`

```bash
cd ch3/part-1
code .

# or

code ch3/part-1
```

```bash
# configure
cmake -B build -G "Ninja Multi-Config"
# build
cmake --build build
# run
./build/Debug/minimal-cmake_game-of-life
```
